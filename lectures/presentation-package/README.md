# Brachial Plexus Lecture -- Presenter Package

Everything you need to deliver the 5-case lecture.

## Files

| File | What it is | How to use it |
|---|---|---|
| **Brachial-Plexus-Lecture.html** | The slide deck (opens in any browser) | Double-click. Press Space / ArrowRight to advance. Esc to exit. Click each "Start Case N" slide to begin that case. |
| **Case-Contents-Guide.pdf** | 8-page annotated guide. For every case: the differentials the system asks about, why each exam / NCS / EMG finding is what it is. | Print it. Use it to answer resident questions on the fly -- each finding has a one-sentence "why" next to it. |
| **Quick-Cheat-Sheet.pdf** | 4-page condensed reference. One-glance summary of all 5 cases. | Fold it in your pocket. Glance at it between cases for the key numbers. |
| **werewolf.jpeg** | Game-night image for the final slide | Keep this file next to the HTML -- the deck loads it by relative path. |

## Presentation flow

1. Open `Brachial-Plexus-Lecture.html`
2. Intro slides -- prerequisites, PMHx reasoning, localization toolkit
3. For each case (1-5):
   - Show the "Start Case N" slide
   - Walk through the clinical case in the Clinical Exam Lab app (the 5 starred cases in the case grid)
   - Return to the deck and reveal diagnosis + pearls + recap
4. Last slide: Werewolf game night reminder

## 5 cases in order

1. **Severe Shoulder Pain Followed by Weakness** -- Parsonage-Turner Syndrome
2. **Hand Weakness After a Fall** -- Klumpke's Palsy (Lower Trunk)
3. **Severe Arm Weakness after Motorcycle Trauma** -- Root Avulsion with Horner's
4. **Hip Flexion Weakness on Warfarin** -- Retroperitoneal Hematoma / Femoral Neuropathy
5. **Foot Drop with Gluteal Sparing Post-THA** -- Sciatic Nerve Injury

## The starred cases

Inside the app, open the Clinical Correlation Lab -> Patient Case Load. The 5 lecture cases are marked with a gold star and a "LECTURE" tag, and they sort to the top of the grid.
